# Guidance for AI Agents

## Core Principles
1. **Always Update the Log**: Every significant change must be recorded in the root `log.md` and the relevant folder's `log.md`.
2. **Standard Links**: Do NOT use `[[WikiLinks]]`. Use `[Standard Markdown Links](./path.md)`.
3. **Frontmatter**: Ensure every file has correct YAML metadata.
4. **Validation**: Run `npm run validate` before finalizing changes.
