# Defer the tutorial until first real task

Replace the up-front tutorial overlay with contextual tooltips triggered the
first time an admin starts a real task (e.g. inviting a teammate).

## Rationale
Supported by the insight [New admins skip the setup
tutorial](../insights/admins-skip-tutorial.md).

## Suggested measure
First-week activation rate for the [New Admin](../personas/new-admin.md) persona.
